## 4. 函数定义

函数是组织代码的重要方式：

<InteractiveCodeBlock
  code={`# 基本函数定义
def greet(name):
    return f"你好, {name}!"

# 带默认参数的函数
def introduce(name, age=18, city="北京"):
    return f"我是{name}，今年{age}岁，来自{city}"

# 可变参数函数
def calculate_sum(*numbers):
    total = 0
    for num in numbers:
        total += num
    return total

# 测试函数
print(greet("小明"))
print(introduce("小红", 20, "上海"))
print(introduce("小华"))  # 使用默认值

result = calculate_sum(1, 2, 3, 4, 5)
print(f"求和结果: {result}")

# Lambda函数
multiply = lambda x, y: x * y
print(f"3 * 4 = {multiply(3, 4)}")`}
/>

## 8. 类和对象

面向对象编程的基础：

<InteractiveCodeBlock
  code={`# 定义类
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def greet(self):
        return f"你好，我是{self.name}，今年{self.age}岁"
    
    def have_birthday(self):
        self.age += 1
        return f"{self.name}过生日了，现在{self.age}岁"

# 创建对象
person1 = Person("小明", 20)
person2 = Person("小红", 18)

print(person1.greet())
print(person2.greet())

print(person1.have_birthday())

# 继承示例
class Student(Person):
    def __init__(self, name, age, major):
        super().__init__(name, age)
        self.major = major
    
    def study(self):
        return f"{self.name}正在学习{self.major}"
    
    def greet(self):
        return f"你好，我是{self.name}，{self.age}岁，专业是{self.major}"

# 测试继承
student = Student("小华", 19, "计算机科学")
print(student.greet())
print(student.study())`}
/>
